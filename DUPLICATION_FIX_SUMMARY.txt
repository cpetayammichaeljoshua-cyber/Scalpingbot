================================================================================
DEDUPLICATION MECHANISM (5-MIN TTL)
================================================================================

HASH FORMAT:
  {symbol}_{direction}_{entry_price:.6f}
  Example: "BTCUSDT_BUY_50000.123456"

CACHE ENTRY STRUCTURE:
  {hash} → timestamp
  If (now - timestamp) > 300 seconds: Entry expires

DEDUP FLOW:
  1. Signal generated in scan_for_signals()
  2. _is_signal_duplicate() checks cache
  3. If in cache AND not expired: Skip signal (return True)
  4. If NOT in cache: Mark as sent (add to cache) AND add to signals list
  5. In auto_scan_loop: Format and send to channel
  6. No re-checking during send (already deduplicated)
